# 個別指導塾 授業管理システム

## セットアップ

```bash
cd juku-system

# パッケージインストール（初回のみ）
pip3 install flask flask-login flask-wtf sqlalchemy werkzeug

# 起動（DBと初期管理者アカウントが自動作成されます）
python3 app.py
```

ブラウザで http://localhost:5050 にアクセス。

## 初期ログイン情報

| ロール | ユーザー名 | パスワード |
|--------|-----------|-----------|
| 管理者 | admin     | admin1234 |
| 先生（サンプル） | yamada / sato / tanaka | pass1234 |

## 機能一覧

- **ログイン/ログアウト**（管理者・先生の2ロール）
- **ダッシュボード**（授業回数統計・月別グラフ）
- **報告書提出**（生徒名サジェスト機能付き）
- **報告書一覧・詳細・編集**
- **管理者：** 先生管理・生徒管理・担当割当・ユーザー管理・未提出アラート
